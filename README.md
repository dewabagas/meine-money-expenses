1. Clone the repository :

```bash
git clone https://github.com/dewabagas/meine-money-expenses.git
cd meine-money-expenses
```
2. Install dependencies :
```bash 
flutter pub get 
```
2. Run the app : 
```bash 
flutter run 
```
